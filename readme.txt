Інструкція для користувачів:

Плагін створює додаткову вкладку в адмінці яка називається "Popups" https://app.screencast.com/EjALIhzcskr1Z
Ця вкладка має 2 додаткові вкладки: "Add popup" та "Popups settings"

На сторінці "Popups" можна бачити список створених попапів
На сторінці "Add popup" можна створити новий попап
Сторінка "Popups settings" містить загальні налаштування для всих попапів

Детальніше про кожну сторінку:
    1. "Popups"
        Це список всих створених попапів (якщо на сайті декілька мов, то це всі попапи, створені на вибраній мові)
        У кожного попапу є свій заголовок (https://app.screencast.com/GNr2igkNoRtQY). Ці заголовки використовуються тільки для ідентифікації попапа в адмінці. Ця назва нікуди не записується, тому називати їх можна як завгодно

        Також з-під цієї вкладки можна редагувати створені попапи. При наведенні на потрібний попап, під заголовком з'являться декілька кнопок. Одна з них це Edit (https://app.screencast.com/x4aaXb6Jwqdiv)
        Після нажаття на Edit ми перейдемо на сторінку редагування попапу

        1.1 Сторінка редагування попапу
            На цій сторінці ви знайдете 3 групи налаштувань: базові, для типу подій і для типу підписок.

            "Base settings" https://app.screencast.com/d6wjQNlrRHkgq
            - "Popup type": Виберіть тип попапу (Event informing - повідомлення про подію; Subscription - прохання підписатись з формою).
            - "Background image": Завантажте фон для вашого попапу.
            - "Colors": Виберіть потрібний колір. Top text color - колір заголовку і підзаголовку; Button color - колір кнопки; Button text color - колір тексту кнопки
            - "Button text": Введіть текст для кнопки.

            "Subscription type settings" (якщо вибрано) https://app.screencast.com/Fp0NlybAxSRMD
            - "Placeholders": Введіть відповідний текст.

            "Event informing type settings" (якщо вибрано) https://app.screencast.com/i1kP4CX6SSYyq
            - "Link to event registration": Вставте посилання.
            - "Show date block": Активуйте, якщо потрібно показувати дату події. Якщо вибрано, то з'являться додаткові поля: "Event date" та "Event time"
            - "Event date": Введіть дату. Це текстове поле
            - "Event time": Виберіть час.

    2. "Popups settings"
        Налаштування попапу події (https://app.screencast.com/NwYRQy5Dlpqit)
        - "Active event popup": Виберіть зі списку потрібний попап, який буде показуватись як основний попап типу події.
        - "Posts, where show event popup": Виберіть сторінки, на яких буде показаний попап. Щоб вибрати декілька, потрібно затиснути Ctrl
        - "Pages, where show event popup": Виберіть пости, на яких буде показаний попап. Щоб вибрати декілька, потрібно затиснути Ctrl
        - "Show on archive pages": Активуйте, якщо потрібно показувати попап на таких сторінках як Новини.

        Налаштування попапу підписки (https://app.screencast.com/9elsRNVSH1uLO)
        - "Active subscription popup": Виберіть зі списку потрібний попап, який буде показуватись як основний попап типу підписки.
        - "Posts, where show subscription popup": Виберіть сторінки, на яких буде показаний попап. Щоб вибрати декілька, потрібно затиснути Ctrl
        - "Pages, where show subscription popup": Виберіть пости, на яких буде показаний попап. Щоб вибрати декілька, потрібно затиснути Ctrl
        - "Show on archive pages": Активуйте, якщо потрібно показувати попап на таких сторінках як Новини.

        Інші налаштування (https://app.screencast.com/Leizc04aXbPfd)
        - "Time offset (in sec)": Введіть затримку показування попапу в секундах.
        - "Mails to which to send data from the form (comma-separated list)": Введіть пошти, на які потрібно відправляти лист після відправки форми з попапу підписки. Список пошт повинен бути записаний через кому.

Деякі інші моменти:
    * Попапи автоматично зникають після закриття і більше не будуть показуватись користувачеві, який це зробив.
      Також вони більше не будуть показуватися після відправки форми та після переходу користувачем на сторінку реєстрації на подію.

    * Для того щоб заголовок і підзаголовок попапу показувався корректно, в контенті папапу заголовок повинен буди типу "Heading 2", а підзаголовок повинен мати тип Paragraph.
